# Dashboard Export - Quick Start Guide

## Download Your Data as Excel

The Finance Tracker allows you to export all your deposits and snapshots as a professional Excel file in just one click.

## How to Download

### Option 1: Header Button (Fastest)
1. Click the **"📥 Download Excel"** button in the top-right corner
2. Wait for the file to download
3. Success message appears confirming the download

### Option 2: Dashboard Card
1. Go to Dashboard
2. Click the **"Download Excel"** card
3. File automatically downloads

## What You Get

Your Excel file includes **3 sheets**:

### Sheet 1: Summary
Quick overview of your export:
- Export date and time
- Your name
- Total deposits count
- Total snapshots count

### Sheet 2: Deposits
Complete list of all deposits with:
- Bank name
- Account holder
- Deposit amount
- Interest rate
- Maturity date
- Account status
- And 11 more columns...

### Sheet 3: Snapshots
Portfolio snapshots with:
- Snapshot date
- Cash balance
- Savings
- Fixed Deposits (FD)
- Recurring Deposits (RD)
- PPF, EPF, NPS accounts
- Emergency fund
- Total portfolio value

## File Details

**Filename Format**: `FinanceTracker_YourName_2025-12-05.xlsx`

**File Type**: Excel 2007+ (.xlsx)

**Size**: Usually 50-500 KB

**Compatibility**:
- ✅ Microsoft Excel
- ✅ Google Sheets
- ✅ LibreOffice
- ✅ Apple Numbers

## Troubleshooting

### "Button not working?"
1. Check internet connection
2. Ensure you're logged in
3. Try refreshing page
4. Check browser console for errors

### "File won't download?"
1. Check browser download settings
2. Disable pop-up blocker
3. Try different browser
4. Check disk space

### "File won't open?"
1. Try different Excel application
2. Download again
3. Check file size > 0 bytes
4. Try opening with Google Sheets

### "Missing data?"
1. Ensure deposits/snapshots exist
2. Create test data first
3. Check API is working
4. Try exporting again

## FAQ

**Q: How often can I download?**
A: Unlimited times! Download whenever you need.

**Q: Can I download other users' data?**
A: No. Only your own data due to security/privacy.

**Q: What if I have 10,000 deposits?**
A: Current limit is 1000. Contact support for larger exports.

**Q: Can I schedule automatic exports?**
A: Not yet, but planned for future version.

**Q: Is the file encrypted?**
A: File is standard Excel. Use Excel password protection if needed.

**Q: Can I edit the exported file?**
A: Yes! It's your Excel file. Edit as needed.

**Q: How long does download take?**
A: Usually < 5 seconds for typical data.

**Q: Does export include matured deposits?**
A: No. Only active deposits. Matured are in separate archive.

**Q: Can I export in other formats?**
A: Currently Excel only. CSV/PDF planned for future.

**Q: Is my data safe?**
A: Yes! Processed locally on your computer. Not stored on server.

**Q: What if download fails?**
A: Error message will appear. Try again or check connection.
